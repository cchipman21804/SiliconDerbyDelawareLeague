Explain it all here...

php -- contains PHP scripts for the Apache web server that need to be copied (via sudo cp) to /var/www/html
       if using keyboard commands, rename (mv) index-js.php to index.php
js --- contains Javascript (script.js) to interpret keyboard commands for index-js.php (needs to be copied to /var/www/html)
py --- Python scripts for various functions
txt -- non-volatile storage for current command & current speed
